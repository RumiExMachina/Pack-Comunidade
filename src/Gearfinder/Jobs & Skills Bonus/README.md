# Gearfinder SRC - Jobs & Skills Bonus
---

## Descrição

Este SRC permite criar bônus para classes e skills. Dando a administradores a possibilidade de adicionar efeitos passivos de Item Bonus diretamente no Job ou Skill. Uma versão modificada de outro sistema disponível na Internet.
